# /morning Workflow

Use this workflow every morning to align the codebase activity with your Taskade GTD system.

## 1. Context Retrieval
- Scan the last 4 days of conversation history (current day + 3 prior).
- Sources:
  - Antigravity Chat Transcripts (via system context).
  - Claude Code Plugin logs (if available as open editors or files in workspace).

## 2. Extraction Logic
- Identify all tasks discussed, implemented, or postponed.
- Filter for tasks that are **NOT DONE** or require follow-up.
- Extract:
  - Task Title
  - Priority Level (Focus, Priority #1, etc.)
  - Related Project/Context (e.g., #Drivenlytics, #BAV)

## 3. Taskade Integration
- Match "Not Done" tasks to the **Drivenlytics / GTD** folder project: `gXTzxLUDtKTE7dH7`.
- For the Top 3 identified priorities:
  - Present to user for approval.
  - Upon approval:
    - Break into 15-minute sub-tasks.
    - Create tasks in the project under the appropriate header (`⚡️ Focus:` or `⚡️ PRIORITY #1`).
    - Assign due dates/times (Taskade event sync).

## 4. Execution Commands
// turbo
1. Run `python3 scripts/morning_sync_04_02_2026.py --review` to generate the summary.
2. Review the proposed Top 3.
// turbo
3. Run `python3 scripts/morning_sync_04_02_2026.py --execute` to push to Taskade after approval.
